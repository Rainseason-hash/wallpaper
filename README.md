壁纸小程序 Web 版
一款基于 UniApp 开发的高颜值壁纸小程序 Web 版本，支持多种分类浏览、高清壁纸预览与下载。
🌟 在线演示
https://rainseason-hash.github.io/wallpaper/
✨ 功能特点
🎨 丰富分类：多种主题分类，轻松找到心仪壁纸
🖼️ 高清预览：支持壁纸高清预览，查看细节
⬇️ 一键下载：简单操作即可下载壁纸到本地
📱 响应式设计：适配各种屏幕尺寸，手机电脑都能完美体验
🔍 搜索功能：快速搜索想要的壁纸主题
🛠️ 技术栈
框架：UniApp
语言：Vue.js
样式：CSS/SCSS
构建工具：HBuilderX
🚀 本地运行
克隆本仓库
bash
git clone https://github.com/Rainseason-hash/wallpaper.git
cd wallpaper
使用 HBuilderX 打开项目
运行到浏览器
在 HBuilderX 中点击「运行」→「运行到浏览器」→ 选择浏览器
